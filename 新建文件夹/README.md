# 民航综合业务平台部署指南

## 快速开始

### 第一步：注册GitHub账号
1. 访问 https://github.com
2. 点击 "Sign up" 注册新账号
3. 完成邮箱验证

### 第二步：创建仓库
1. 登录GitHub后，点击右上角 "+" → "New repository"
2. 仓库名填写：`civil-aviation-platform`
3. 描述可填："民航综合业务平台"
4. 选择 "Public"（公开）
5. 勾选 "Add a README file"
6. 点击 "Create repository"

### 第三步：上传文件
1. 在仓库页面，点击 "Add file" → "Upload files"
2. 将以下3个文件拖拽到上传区域：
   - index.html
   - login.html
   - README.md
3. 点击 "Commit changes"

### 第四步：开启GitHub Pages
1. 进入仓库设置：Settings → Pages
2. 在 "Source" 选择 "Deploy from a branch"
3. 在 "Branch" 选择 "main" 分支，文件夹选择 "/ (root)"
4. 点击 "Save"
5. 等待几分钟，显示 "Your site is live at https://你的用户名.github.io/civil-aviation-platform/"
